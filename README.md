# AI求职助手

面向 0 AI产品经验转行 AI 产品经理求职者的岗位 JD 分析助手。

用户可以从 BOSS 直聘等招聘平台复制 AI 产品经理岗位正文，粘贴到系统中进行分析。系统会基于岗位 JD 和用户画像，生成 Top 关键词、候选人核心能力要求、能力分析、面试重点、岗位洞察和求职建议，帮助传统产品经理更高效地理解 AI PM 岗位要求。

## 在线体验

(https://ai-job-assistant-puce.vercel.app)

## 目标用户

面向希望转型 AI 产品经理的求职者，尤其是：

- 有传统产品经理经验
- 缺少 AI 产品项目经验
- 过往做过工具类、出行类、内容类、SaaS、B 端等产品
- 希望通过真实 JD 理解 AI PM 市场要求

## 核心功能

### 1. 岗位详情粘贴分析

- 支持粘贴 BOSS 直聘等平台的岗位正文
- 支持一次粘贴多个岗位
- 最多支持一次分析 50 个岗位
- 不支持直接粘贴岗位 URL，避免招聘平台反爬和验证页导致分析不稳定

### 2. 数据概览

- 岗位数量
- 平均薪资

### 3. Top 关键词

从 JD 中提取 AI 产品经理相关关键词，并统计出现次数。

示例关键词：

- RAG
- Prompt
- Agent
- LLM
- 问答
- 模型评测
- 数据分析
- A/B 测试
- 项目推进
- 商业化
- B 端
- C 端

### 4. 候选人核心能力要求

将 JD 中的关键词映射为能力域，帮助用户理解岗位真正要求什么能力。

能力域包括：

- RAG / 知识库
- Prompt 工程
- Agent 产品设计
- AI 应用设计
- 模型评测
- 数据分析
- 产品方法论
- 项目推进
- 商业化理解
- 增长产品
- B 端产品
- C 端产品

### 5. 用户画像

支持编辑用户画像字段：

- 工作年限
- 学历背景
- 有无产品经验
- 过往产品类型
- 熟悉的技术
- 擅长的技能

用户画像会参与能力分析、面试重点和求职建议生成。

### 6. 能力分析

结合 JD 能力要求和用户画像，判断每项能力属于：

- 优势能力
- 可迁移能力
- 重点补齐

系统会同时给出优先级、判断原因和建议动作。

### 7. 面试重点

基于 JD 高频能力和用户能力差距，生成面试准备重点：

- 必备问题
- 案例包装
- 短板补救
- 临时补课

### 8. 岗位洞察

总结本批次岗位的共性要求，例如：

- 高频能力趋势
- AI 专项能力和传统产品能力占比
- 重点能力在面试中的可能追问方向
- 薪资解析情况

### 9. 求职建议

结合用户画像和岗位要求，生成求职准备建议，例如：

- 简历开头如何表达
- 过往经验如何迁移到 AI 产品语境
- 需要优先补齐哪些能力
- 面试中应该准备什么案例

### 10. BOSS 岗位复制辅助扩展

项目中包含一个 Chrome 扩展 MVP：

```text
browser-extension/
```

用于辅助从 BOSS 直聘岗位页面复制当前可见的岗位信息。

扩展能力：

- 复制当前岗位
- 加入暂存
- 导出暂存
- 清空暂存
- 最多暂存 50 个岗位

## 技术架构

```text
用户浏览器
  |
  | 打开网页
  v
Vercel CDN
  |
  | 返回静态资源
  v
React 单页应用
  |
  | 本地执行 JD 分析逻辑
  v
分析结果看板
```

当前项目没有后端、没有数据库、没有登录系统，也不会保存用户粘贴的 JD 内容。
