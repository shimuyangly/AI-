# AI求职助手 - BOSS岗位复制器

这是一个 Chrome Manifest V3 插件，用于在 BOSS 直聘岗位详情页辅助复制当前页面已经展示出来的岗位文本。

它不请求 BOSS 接口，不批量打开页面，不绕过登录、验证或反爬机制，只整理当前可见页面文本。

## 安装

1. 打开 Chrome：`chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本目录：`browser-extension`

## 使用

1. 打开 BOSS 直聘岗位详情页
2. 页面右下角会出现“AI求职助手”面板
3. 可执行：
   - `复制当前岗位`：整理并复制当前岗位
   - `加入暂存`：把当前岗位加入暂存区
   - `导出暂存`：把暂存岗位合并复制，最多 50 个
   - `清空暂存`：清空暂存区

## 导出格式

```text
岗位名称：xxx
薪资：xxx
岗位介绍：
xxx

岗位名称：xxx
薪资：xxx
岗位介绍：
xxx
```

复制后可直接粘贴到 AI求职助手中分析。
